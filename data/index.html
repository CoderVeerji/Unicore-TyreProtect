Installation Guide

Run Claude
on your machine

Step-by-step guide to install and launch Claude Code locally.
Works on macOS & Windows.

macOS steps
1
Install Claude Code via CLI
Run this single command in your Terminal. It downloads and installs Claude Code automatically.
Terminal
$ curl -fsSL https://claude.ai/install.sh | bash
💡 Open Terminal via Spotlight (⌘ Space) → type "Terminal" → press Enter.
2
Download & Sign In to Ollama
Ollama lets you run large language models locally. Download it and sign in with Google.
🦙
ollama.com/download
Download the macOS app — requires macOS 12 or later
↗
ℹ️ After installing, open Ollama from Applications and log in using your Google account when prompted.
3
Launch Claude via Ollama
Pull and run the Claude model through Ollama's CLI.
Terminal
# Pull and launch Claude model
$ ollama run claude
⚡ First run downloads the model. This may take a few minutes depending on your connection.
4
Start Claude Code
You're ready! Launch Claude Code from your terminal in any project directory.
Terminal
# Navigate to your project first (optional)
$ cd ~/your-project

# Launch Claude Code
$ claude
Windows steps
1
Install Claude Code via PowerShell
Open PowerShell as Administrator and run the install command below.
PowerShell (Admin)
# Option A — PowerShell installer
PS> irm https://claude.ai/install.ps1 | iex

# Option B — via npm (Node.js required)
PS> npm install -g @anthropic-ai/claude-code
ℹ️ To open PowerShell as Admin: Press Win + X → click "Windows PowerShell (Admin)" or "Terminal (Admin)".
⚠️ If you get a script execution error, first run:
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
2
Download & Sign In to Ollama
Ollama has a native Windows installer. Download it and sign in with Google.
🦙
ollama.com/download/windows
Windows installer (.exe) — requires Windows 10/11 (64-bit)
↗
ℹ️ After install, Ollama runs in the system tray. Open it and log in with your Google account.
3
Launch Claude via Ollama
Open a new PowerShell or Command Prompt window and pull the Claude model.
PowerShell / CMD
# Pull and launch Claude model
PS> ollama run claude
⚡ First run downloads the model — grab a coffee! Subsequent launches are instant.
4
Start Claude Code
Open a new PowerShell or CMD window inside your project folder and run claude.
PowerShell / CMD
# Navigate to your project first (optional)
PS> cd C:\Users\YourName\your-project

# Launch Claude Code
PS> claude
💡 Pro tip: Right-click any folder in Explorer → "Open in Terminal" to skip the cd step!
🚀
You're all set!

Claude Code is running locally on your machine.
Navigate to any project folder and type claude to start coding with AI.

$ claude█
